# Bookify

Bookify is an online platform where users can explore a wide variety of books across different genres. With a user-friendly interface, Bookify allows readers to discover, read, and enjoy a curated selection of literature.

## Live Demo

Check out the live version here: [Bookify](https://bookify-store.vercel.app/){:target="_blank"}

## Features

- **Free Access**: Users can sign up, read, and download books for free.
- **Browse and Search Books** by genre and category.
- **User Registration and Login** using Firebase Authentication.
- **Book Details** with ratings.
- **Responsive Design** for mobile and desktop.

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database**: Firebase for authentication, MongoDB for feedback
- **Deployment:** Vercel

## Usage

Sign up, log in, and explore the wide range of books available for free. Users can also provide feedback on the platform.
